I’m excited to share my Machine Intelligence course project: **Stress Level Classifier** 🧠

This project uses machine learning to classify short written text responses into **Low**, **Medium**, or **High** stress levels.

What the project includes:

✅ Dataset collection and preprocessing  
✅ Text cleaning, duplicate handling, and class balancing  
✅ TF-IDF feature extraction  
✅ Training and comparison of 5 ML models  
✅ Performance evaluation using Accuracy, Macro Precision, Macro Recall, and Macro F1  
✅ A Streamlit interface for live prediction  

The five models compared were:

- Multinomial Naive Bayes
- Logistic Regression
- Linear SVM
- K-Nearest Neighbors
- Decision Tree

The best-performing model was **Logistic Regression**, achieving the strongest Macro F1-score across the three stress classes.

This project helped me better understand the complete ML workflow: from collecting and preparing data to training models, comparing results, and deploying a simple user interface.

Tech stack: **Python, pandas, scikit-learn, TF-IDF, Streamlit, Matplotlib**
